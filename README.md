# Visual Resume

This repository contains the source for Martinho Hoffman's visual, interactive resume.

The live version is available at [martinhohoff.github.io/visual-resume](https://martinhohoff.github.io/visual-resume/).

For the complete resume and up-to-date professional history, see [LinkedIn](https://www.linkedin.com/in/martinhohoff/).

## What This Repo Contains

- [index.html](/Users/martinhohoff/code/martinhohoff/visual-resume/index.html): the static interactive resume page
- [profile.json](/Users/martinhohoff/code/martinhohoff/visual-resume/profile.json): the structured resume data rendered by the page

## Local Preview

Open [index.html](/Users/martinhohoff/code/martinhohoff/visual-resume/index.html) in a browser, or serve the repository locally with any simple static file server.
